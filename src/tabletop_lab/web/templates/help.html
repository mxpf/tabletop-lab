Help

What Tabletop Lab Is

Tabletop Lab is a local playtest workspace for small tabletop card and token games. It stores rules, runs deterministic bot simulations, saves prior runs, and compares metrics.

Create a Game

Use New Game to create a record with a name, description, notes, and rules text. Black Ledger is preloaded as a built-in playable module.

Paste or Upload Rules

You can paste rules into a textarea or upload a PDF. PDF text is extracted and stored with the game record.

Pasted or uploaded rules are stored for reference. To simulate a new game, a game module still needs to be implemented in code.

Run a Black Ledger Simulation

Open Black Ledger, choose a variant, choose bots for Player 0, Player 1, and Player 2, set the number of games, and start the run. The run detail page polls progress.

Read the Metrics

Run summaries include average turns, end condition rates, average scores, win rates, action counts, Call success rate, Burn count, and Furnace outcomes.

Compare Prior Tests

Open Compare Runs from a game page, select completed runs, and review the table side by side.

CLI

python3 scripts/run_game.py --game black_ledger --variant base_3p --seed 42 --transcript
python3 scripts/run_simulation.py --game black_ledger --variant five_numerals_starting_ledger_3 --games 1000 --progress
python3 scripts/compare_variants.py base_3p five_numerals_starting_ledger_3 --games 5000 --progress
